# Online Reservation System

## Description
The Online Reservation System is a Java Swing and SQLite based application that allows users to log in, book train tickets, generate unique PNR numbers, and cancel reservations using the PNR number.

## Features
- User Login Authentication
- Train Reservation Form
- Automatic Train Name Population
- PNR Generation
- SQLite Database Storage
- Ticket Cancellation Using PNR
- Booking Confirmation Dialog
- Input Validation

## Technologies Used
- Java
- Swing
- JDBC
- SQLite
